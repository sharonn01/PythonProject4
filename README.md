### 2. **Student Performance Analytics** (`proj_week4.ipynb`)
Educational data analysis project analyzing student academic performance, attendance, and overall academic health.

#### Features:
- **Data Generation**: Synthetic student data with realistic attributes
- **Performance Metrics**: Analysis of marks across multiple subjects (Maths, Science, English)
- **Attendance Tracking**: Student attendance pattern analysis
- **Missing Data Handling**: Data cleaning and imputation techniques
- **Statistical Insights**: Grade distribution and performance statistics
- **Visualizations**: Academic performance trends and correlations

#### Technologies Used:
- Python 3
- pandas - Data manipulation
- NumPy - Numerical operations
- Faker - Synthetic data generation
- Matplotlib - Plotting
- Seaborn - Statistical visualization

#### Key Metrics:
- Subject-wise performance analysis
- Attendance correlation with grades
- Average score calculations
- Internal marks assessment
